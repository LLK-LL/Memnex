# Chinese Academic Style

## Purpose

Use this reference to polish Chinese scientific writing, reduce AI-like phrasing, and make thesis/report prose concise, precise, and defensible.

## Core Style

- Prefer short subject-verb-object sentences.
- Put the concrete technical object before abstract evaluation.
- Use stable terminology. Do not rotate synonyms for variety.
- Keep paragraphs centered on one claim.
- Use transitions to show logic, not decoration.

## Evidence-Calibrated Verbs

Use cautious verbs for midterm-stage work:

- 琛ㄦ槑
- 璇存槑
- 鍒濇楠岃瘉
- 鍙嶆槧
- 鏈夊姪浜?- 涓哄悗缁?..鎻愪緵鍩虹

Use stronger verbs only with completed evidence:

- 璇佹槑
- 鏄捐憲浼樹簬
- 鏈夋晥闄嶄綆
- 绋冲畾鎻愬崌
- 绯荤粺楠岃瘉

## Paragraph Pattern

For technical paragraphs, use:

1. Topic sentence: the claim or purpose
2. Technical detail: method, data, module, or experiment
3. Evidence or rationale
4. Transition to the next point

## AI-Like Pattern Checks

Flag and revise:

- repeated "棣栧厛/鍏舵/鏈€鍚? without real hierarchy
- dense strings of abstract nouns
- every paragraph ending with "鍏锋湁閲嶈鎰忎箟"
- inflated adjectives: 閲嶅ぇ, 鏄捐憲, 鍏ㄩ潰, 娣卞叆, 鏈夋晥, 鍒涙柊鎬?- generic conclusions without data
- English-style long sentences translated into stiff Chinese

## Before Returning Polished Text

Check that every changed sentence preserves the original technical meaning, citations, quantities, and hedging level. Do not strengthen claims while polishing style.
